ID: {message_id}") except Error as e: print(f"Send error => code: {e.code}, message: {e.message}") if e.code == ResponseCode.ErrUnprocessableEntity.value: for field in e.message: print(f"Field: {field} , Errors: {e.message[field]}") except HTTPError as e: # http error like network error, not found, ... print(f"Error handled => code: {e}")
